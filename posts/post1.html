Approximating a Definite Integral Using Uniform(0,1) Random Numbers

Numerical integration is traditionally introduced using deterministic techniques such as the trapezoidal rule or Simpson’s rule. But there exists a fundamentally different idea:

Replace the integral with an expectation.
Estimate the expectation using randomness.

This idea forms the foundation of Monte Carlo Integration.

1. The Core Idea

Suppose we want to compute:

\[ I = \int_0^1 f(x)\,dx \]

Let

\[ U_1, U_2, \dots, U_n \sim \text{Unif}(0,1) \] \[ \mathbb{E}[f(U)] = \int_0^1 f(x)\,dx \]

By the Law of Large Numbers,

\[ \frac{1}{n}\sum_{i=1}^n f(U_i) \longrightarrow \int_0^1 f(x)\,dx \]

The Monte Carlo estimator is:

\[ \hat I_n = \frac{1}{n} \sum_{i=1}^n f(U_i) \]

2. General Interval [a,b]

\[ I = \int_a^b f(x)\,dx \]

Generate

\[ U_i \sim \text{Unif}(0,1) \]

Transform

\[ X_i = a + (b-a)U_i \] \[ \hat I_n = (b-a)\frac{1}{n}\sum_{i=1}^n f(X_i) \]

3. Geometric Interpretation

Think of the integral as the average height of the function multiplied by the interval width. Monte Carlo estimates the average height using random samples.

4. Algorithm

  1. Generate \(U_1, \dots, U_n \sim \text{Unif}(0,1)\)
  2. Compute \(X_i = a + (b-a)U_i\)
  3. Compute \[ \hat I_n = (b-a)\frac{1}{n}\sum_{i=1}^n f(X_i) \]

5. Python Implementation

import numpy as np

def monte_carlo_integral(f, a, b, n=10000):
    U = np.random.uniform(0, 1, n)
    X = a + (b - a) * U
    return (b - a) * np.mean(f(X))

f = lambda x: x**2

estimate = monte_carlo_integral(f, 0, 1, 100000)
print("Monte Carlo Estimate:", estimate)
print("True Value:", 1/3)

6. Statistical Properties

Unbiasedness

\[ \mathbb{E}[\hat I_n] = I \]

The estimator is unbiased.

Variance

\[ \text{Var}(\hat I_n) = \frac{(b-a)^2}{n} \text{Var}(f(U)) \]

Standard error behaves as \(O(1/\sqrt{n})\).

7. Central Limit Theorem

\[ \sqrt{n}(\hat I_n - I) \xrightarrow{d} \mathcal{N}(0,\sigma^2) \]

Approximate 95% confidence interval:

\[ \hat I_n \pm 1.96 \cdot \frac{\hat\sigma}{\sqrt{n}} \]

8. A Famous Example: Estimating π

\[ \pi = 4 \cdot \mathbb{P}(X^2 + Y^2 \le 1) \]

Generate random points in the unit square and compute the proportion inside the quarter circle. Multiply by 4 to approximate π.

Final Thoughts

A definite integral is simply an expectation under a uniform distribution. Monte Carlo replaces area computation with averaging randomness.
